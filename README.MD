### O que eu aprendi

Neste projeto aprofundei consideravelmente na utilizacao do box shadow e posicinamento de imagens.





``` css
.card{
    width: 300px;
    height: 500px;
    background-color:  white;
    padding: 10px;
    padding-top: 20px;
    margin:  auto;
    margin-top: 120px;
    border-radius: 5%;
    box-shadow: 5px 5px 20px hsl(218, 44%, 22%);
```
